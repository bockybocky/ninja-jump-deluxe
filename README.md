# 🥷 Ninja Jump Deluxe — 忍者跳躍豪華版

一款超豪華 HTML5 跑酷遊戲，直接在瀏覽器中遊玩！

## 🎮 [立即遊玩](https://bockybocky.github.io/ninja-jump-deluxe/)

## 操控方式

| 動作 | 鍵盤 | 手機 |
|------|------|------|
| 跳躍 | `空白鍵` / `↑` | 點擊畫面 |
| 二段跳 | 空中再按一次 | 空中再點一次 |
| 蹲下 | `↓` | 下滑 |

## ✨ 遊戲特色

- 🏙️ 視差捲動城市夜景背景
- 🌟 日夜循環 & 星空閃爍
- 💥 粒子特效（揚塵、爆炸、金幣閃光）
- 🌵 多種障礙物（仙人掌、岩石、飛鳥）
- 🪙 金幣收集 & Combo 連擊系統
- ⭐ 三種道具（護盾、磁鐵、雙倍分數）
- 🔊 Web Audio API 合成音效
- 📱 手機觸控完整支援
- 🏆 最高分紀錄（localStorage）
- 📈 5 段難度遞增（EASY → HELL 🔥）
